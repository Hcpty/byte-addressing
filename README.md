A note about Page Table.

### 字节地址的数据结构和字节地址的计算公式

一个页中包含2<sup>p</sup>个字节，每一个字节都有一个字节地址，字节地址的数据结构中包含两个字段：
- 该字节所在的页的页号，根据页号可以计算出页地址，页地址 = 页号 * 2<sup>p</sup>。
- 该字节在该页中的偏移字节数。

字节地址的计算公式：字节地址 = 页地址 + 偏移字节数。

### 从逻辑页地址到物理页地址的映射

一个磁盘文件可以包含很多页，每一页都有一个磁盘地址，当把该文件加载到主存中后，每一页都有一个主存地址。
